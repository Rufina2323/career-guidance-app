# Career Prediction Service - MLPO Project

## Описание проекта
Web-приложение для прогнозирования профессий на основе навыков, интересов и академических данных студентов.  
Пользователи могут загружать данные, получать предсказания, просматривать историю запросов и баланс кредитов. Администратор может пополнять баланс пользователей и просматривать все транзакции.
